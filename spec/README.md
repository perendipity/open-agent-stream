# Spec Index

`/spec` is the normative source of truth.

- `raw-envelope/v1`: immutable capture record
- `canonical-event/v1`: portable event stream
- `analytics-dataset/v1`: portable analytics snapshot dataset and manifest
- `session-model/v1`: stable session identity
- `source-adapter/v1`: source contract
- `sink/v1`: sink contract and checkpoint schema
- `privacy-policy/v1`: redaction policy view schema
